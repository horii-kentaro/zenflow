A-3. 外部API・SaaS

サービス名 用途 料金体系 環境変数キー 備考
Anthropic API（Claude） AIジャーナリング対話・AI週間レポート・パーソナライズドルーティン 従量課金 ANTHROPIC_API_KEY claude-sonnet-4-5-20250514 使用
Stripe Proプラン（月額980円）のサブスクリプション決済 決済手数料 3.6% STRIPE_SECRET_KEY
STRIPE_WEBHOOK_SECRET		 Checkout Session + Webhook
Resend メール認証・サブスク通知メール Free: 月3,000通 RESEND_API_KEY ドメイン認証済み
PostgreSQL ユーザー・気分・ジャーナル・セルフケアデータ永続化 ローカル: Docker / 本番: Vercel Postgres DATABASE_URL Prisma 6 ORM 経由
補足：Anthropic API は AIジャーナリングの対話コーチ（claude-sonnet-4-5-20250514）として使用。Proプランではさらに AI週間レポート生成・パーソナライズドルーティン推薦にも活用。
ANTHROPIC_API_KEY にはダミー値が入っているため、本番運用時は実際のキーに差し替えが必要。

Stripe：フリーミアムモデル（無料 → Pro 月額980円）。Stripe Checkout + Webhook でサブスクリプション管理。未設定時は「決済機能は準備中です」メッセージを表示。

モバイルアプリ：Expo SDK 55 + Dev Client ビルド（Expo Go は使用不可）。iPhone 17 Pro シミュレータ（iOS 26.3）で動作確認。iOS 26 シミュレータは絵文字レンダリング不可のため、全UIで Ionicons / MaterialCommunityIcons を使用。

状態管理：Zustand で plan（free/premium）、todayStatus（mood/selfcare/journal完了状態）、onboardingDone をグローバル管理。タブ間の状態共有（バッジ表示等）に活用。

A-6. 環境変数一覧

環境変数キー 用途 設定場所
DATABASE_URL PostgreSQL 接続URL .env + .env.local
AUTH_SECRET NextAuth.js JWT 署名鍵 .env.local
AUTH_URL NextAuth.js ベースURL .env.local
ANTHROPIC_API_KEY Claude API 認証 .env.local
STRIPE_SECRET_KEY Stripe サーバーサイド認証 .env.local / Vercel
STRIPE_WEBHOOK_SECRET Stripe Webhook 署名検証 .env.local / Vercel
NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY Stripe クライアントサイド公開鍵 .env.local / Vercel
RESEND_API_KEY Resend メール配信 API 認証 .env.local / Vercel
注意：Prisma CLI は .env.local を読めないため、.env にも DATABASE_URL を記載する必要があります。
ローカル開発は docker compose up -d で PostgreSQL を起動してから npm run dev で開発サーバーを起動してください。
